# E-Commerce Application with AI Chatbot\n\nA full-stack e-commerce platform with integrated AI chatbot powered by OpenAI, built with React, Node.js, Express, and MySQL.\n\n## Features\n\n- 🛍️ **Product Management** - Browse, search, and filter products\n- 🛒 **Shopping Cart** - Add/remove items, manage quantities\n- 📦 **Order Management** - Create and track orders\n- 💬 **AI Chatbot** - Real-time chat powered by OpenAI GPT\n- 👤 **User Authentication** - JWT-based secure authentication\n- ⭐ **Product Reviews** - Rate and review products\n- 📱 **Responsive Design** - Mobile-friendly interface\n- 🔐 **Admin Panel** - Manage products and orders\n- 🚀 **Real-time Updates** - Socket.io integration\n\n## Tech Stack\n\n### Backend\n- **Node.js** - Runtime environment\n- **Express.js** - Web framework\n- **Sequelize** - ORM for database\n- **MySQL** - Database\n- **JWT** - Authentication\n- **OpenAI API** - AI chatbot\n- **Socket.io** - Real-time communication\n\n### Frontend\n- **React 18** - UI library\n- **Redux Toolkit** - State management\n- **React Router** - Navigation\n- **Tailwind CSS** - Styling\n- **Axios** - HTTP client\n- **Socket.io Client** - Real-time chat\n- **Vite** - Build tool\n\n## Project Structure\n\n```\n├── backend/\n│   ├── config/\n│   │   └── database.js\n│   ├── models/\n│   │   ├── User.js\n│   │   ├── Product.js\n│   │   ├── Cart.js\n│   │   ├── Order.js\n│   │   ├── OrderItem.js\n│   │   ├── Review.js\n│   │   ├── ChatHistory.js\n│   │   ├── Category.js\n│   │   └── index.js\n│   ├── routes/\n│   │   ├── auth.js\n│   │   ├── products.js\n│   │   ├── cart.js\n│   │   ├── orders.js\n│   │   ├── chatbot.js\n│   │   └── users.js\n│   ├── middleware/\n│   │   └── auth.js\n│   └── server.js\n├── frontend/\n│   ├── src/\n│   ├── index.html\n│   ├── package.json\n│   └── .env.example\n├── .env.example\n├── .gitignore\n└── README.md\n```\n\n## Getting Started\n\n### Prerequisites\n- Node.js (v14 or higher)\n- MySQL (v5.7 or higher)\n- npm or yarn\n\n### Backend Setup\n\n1. Navigate to backend directory:\n```bash\ncd backend\n```\n\n2. Install dependencies:\n```bash\nnpm install\n```\n\n3. Create `.env` file from `.env.example`:\n```bash\ncp .env.example .env\n```\n\n4. Update `.env` with your configuration:\n```env\nDB_HOST=localhost\nDB_USER=root\nDB_PASSWORD=your_password\nDB_NAME=ecommerce_db\nDB_PORT=3306\nNODE_ENV=development\nPORT=5000\nJWT_SECRET=your_secret_key\nJWT_EXPIRE=7d\nOPENAI_API_KEY=your_openai_api_key\nOPENAI_MODEL=gpt-3.5-turbo\nFRONTEND_URL=http://localhost:3000\n```\n\n5. Start the server:\n```bash\nnpm start\n```\n\nServer will run on `http://localhost:5000`\n\n### Frontend Setup\n\n1. Navigate to frontend directory:\n```bash\ncd frontend\n```\n\n2. Install dependencies:\n```bash\nnpm install\n```\n\n3. Create `.env` file from `.env.example`:\n```bash\ncp .env.example .env\n```\n\n4. Update `.env`:\n```env\nVITE_API_URL=http://localhost:5000/api\nVITE_SOCKET_URL=http://localhost:5000\n```\n\n5. Start development server:\n```bash\nnpm run dev\n```\n\nApplication will be available at `http://localhost:5173`\n\n## API Endpoints\n\n### Authentication\n- `POST /api/auth/register` - Register new user\n- `POST /api/auth/login` - Login user\n\n### Products\n- `GET /api/products` - Get all products\n- `GET /api/products/:id` - Get product details\n- `GET /api/products?category=id` - Filter by category\n- `GET /api/products?search=keyword` - Search products\n\n### Cart\n- `GET /api/cart` - Get cart items\n- `POST /api/cart` - Add to cart\n- `DELETE /api/cart/:id` - Remove from cart\n\n### Orders\n- `GET /api/orders` - Get user orders\n- `POST /api/orders` - Create order\n\n### Chatbot\n- `POST /api/chatbot/message` - Send message to AI chatbot\n- `GET /api/chatbot/history/:sessionId` - Get chat history\n\n### User\n- `GET /api/users/profile` - Get user profile\n- `PUT /api/users/profile` - Update profile\n\n## Database Schema\n\n### Users\n- id, firstName, lastName, email, password, phone, address, role, isActive\n\n### Products\n- id, name, description, price, discountPrice, stock, sku, images, category, rating, brand\n\n### Orders\n- id, orderNumber, userId, totalAmount, status, paymentStatus, shippingAddress\n\n### Cart\n- id, userId, productId, quantity, selectedSize, selectedColor\n\n### ChatHistory\n- id, userId, sessionId, userMessage, botResponse, intent\n\n## Configuration\n\n### Environment Variables\n\n**Backend (.env)**\n```\nDB_HOST=localhost\nDB_USER=root\nDB_PASSWORD=root\nDB_NAME=ecommerce_db\nDB_PORT=3306\nNODE_ENV=development\nPORT=5000\nJWT_SECRET=your_jwt_secret_key\nJWT_EXPIRE=7d\nOPENAI_API_KEY=your_openai_api_key\nOPENAI_MODEL=gpt-3.5-turbo\nFRONTEND_URL=http://localhost:3000\nSMTP_HOST=smtp.gmail.com\nSMTP_PORT=587\nSMTP_USER=your_email@gmail.com\nSMTP_PASSWORD=your_email_password\nSTRIPE_SECRET_KEY=your_stripe_secret_key\nSTRIPE_PUBLIC_KEY=your_stripe_public_key\n```\n\n**Frontend (.env)**\n```\nVITE_API_URL=http://localhost:5000/api\nVITE_SOCKET_URL=http://localhost:5000\n```\n\n## Development\n\n### Running Tests\n```bash\n# Backend\ncd backend\nnpm test\n\n# Frontend\ncd frontend\nnpm test\n```\n\n### Build for Production\n\n**Backend**\n```bash\ncd backend\nnpm run build\n```\n\n**Frontend**\n```bash\ncd frontend\nnpm run build\n```\n\n## Deployment\n\n### Using Docker\n\n```bash\ndocker-compose up\n```\n\n### Hosting\n- **Backend**: Deploy to Heroku, Railway, or AWS\n- **Frontend**: Deploy to Vercel, Netlify, or GitHub Pages\n- **Database**: Use managed MySQL service (AWS RDS, DigitalOcean)\n\n## Contributing\n\n1.
